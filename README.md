# 🌐 Multi-Page Website Project

## 📌 Project Overview

This is a simple multi-page website created using basic HTML.
The project includes three main pages: Home, About, and Contact.

---

## 🚀 Features

* 🏠 Home Page with welcome message and services list
* 👤 About Page with mission and team information
* 📞 Contact Page with contact details and form
* 🔗 Navigation bar to switch between pages
* 📄 Simple and clean HTML structure

---

## 🗂️ Project Structure

```
project-folder/
│
├── index.html      # Home Page
├── about.html      # About Page
├── contact.html    # Contact Page
└── README.md       # Project Documentation
```

---

## 🛠️ Technologies Used

* HTML5

---

## 📖 Pages Description

### 🏠 Home Page (index.html)

* Welcome message
* Services list
* Navigation links

### 👤 About Page (about.html)

* Mission statement
* Team information table

### 📞 Contact Page (contact.html)

* Address, email, phone
* Contact form

---

## 💡 How to Run

1. Download or clone the repository
2. Open the project folder
3. Double click on `index.html`
4. Website will open in your browser
